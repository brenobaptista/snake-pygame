# Snake Pygame

> Snake game made using Pygame 2

![Gameplay GIF](https://github.com/brenobaptista/snake-pygame/blob/main/data/gameplay.gif)

## Getting Started

```
Install dependencies:
$ pip install -r requirements.txt

Run game:
$ python snake.py
```

## Building on Windows

```
Install PyInstaller:
$ pip install pyinstaller

Generate the bundle:
$ pyinstaller snake.py --onefile --noconsole â€“-add-data "data;data" --icon=data/icon.ico
```

The executable will be inside the folder `dist`.
